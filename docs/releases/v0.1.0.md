# v0.1.0 - Local-First Capture + Atom Generator Demo

## Highlights

1. Browser extension capture flow to local inbox.
2. Authenticated capture API with validation and URL canonicalization.
3. New atom generator demo (`/api/v1/atoms/generate` + CLI).
4. Parameterized generation: goal, dedupe strength, novelty.
5. Multi-provider support (`mock`, `openai`) with traceable output persistence.

## New Commands

```bash
cd inbox-server
npm run atoms:generate -- --inbox-dir ../inbox --goal inspire --dedupe-strength medium --novelty balanced --count 15 --provider mock --model demo-v1 --temperature 0.7 --days 30
```

## Testing

1. Full suite: `npm test`
2. Includes API/service/CLI tests for atom generation.
3. Real provider smoke test auto-skips unless OpenAI env vars are set.

## Notes

This release is demo-focused. The priority is feature richness and complete functional test coverage.
